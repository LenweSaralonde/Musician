Musician.TRP3E.AddSheetMusicItem (title, encodedSongData[, locale[, icon=Musician.TRP3E.ITEM_ICON[, quantity=1]]])
Add or update existing sheet music item into TRP3 items

Parameters:

  • title (string) Song title
  • encodedSongData (string) Base64-encoded song data
  • locale (string) Item locale. By default, the current realm locale is used. (optional)
  • icon (string) Item icon (default Musician.TRP3E.ITEM_ICON)
  • quantity (int) Item quantity to be added in the main inventory (default 1)

Returns:

    objectID (string)
