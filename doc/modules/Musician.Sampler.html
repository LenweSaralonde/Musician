Musician.Sampler.GetSoundFile (instrument, key)
Get the path to a sound file for an instrument and a key. Also returns the instrument data from Musician.INSTRUMENTS and all other suitable sound file paths for randomization.

Parameters:

  • instrument (int|string|table) MIDI instrument index, instrument name or instrument data
  • key (int) MIDI key

Returns:

  1. soundFile (string|nil) file path to be played
  2. instrumentData (table|nil) from Musician.INSTRUMENTS
  3. soundFiles (table|nil) all possible file paths for this instrument and note, when applicable
